    Disclaimer

    The user is allowed to make 100 requests to Torn's API in a minute.

    The creator of this extension is not responsible if the user:

    • spams requests too much
    • tries to use the wrong API key too many times

    If for some reason you do get your IP blocked because the extension had a bug or glitched or anything - our apologies. Please do let us know of such incidents so we can avoid them in the future.

    API Usage

    • Background
      • User's information, status, cooldown, money, unread events, unread messages and refills. (every 30 seconds (configurable))
        • User's personal stats, networth, stock inventory, ammo, stats, crimes, skills, weapon experience, educations and merit allocation. (every 2 minutes (configurable), when active on Torn)
        • User's attack history. (whenever an attack or defend is detected)
      • Stakeout target's information. (every 30 seconds (configurable), 1 request per stakeout)
      • Faction crime data. (every 15 minutes)
      • Torn's stock data. (every 5 minutes)
      • Torn's items, point values, properties, honors, medals and educations. (daily)
    • Individually
      • Market popup: 1 request per item
      • Bank: 1 request per visit for the interest
      • Display cabinet worth: 1 request per visit for the worth
      • Bazaar worth: 1 request per visit
      • Company Special - muggable cash: 1 request per special usage, a 2nd if that user is in a clothing store
      • NNB: 1 request to YATA and 1 request to TornStats
      • Last Action - factions: 1 request per visit
      • Last Action - companies: 1 or 2 requests per visit
      • Profile Box - stats: 1 request per profile visit
      • Profile Box - spy report: 1 request to YATA and 1 request to TornStats per profile visit
      • Stats Estimate: 1 request per estimate
    • Cache
      • NNB: 1 hour
      • Bank interest: till new day
      • Profile Box - stats: 6 hours
      • Profile Box - spy report: 1 day, or 6 hours if the result is over 31 days old, or 1 hour if there is no result
      • Stats Estimate: 7 days, or 31 days if 'over 200' or inactive for over 180 days, or 1 day if not able to calculate it
